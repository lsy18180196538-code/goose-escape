# 🦆 鹅鹅快跑 - 埃及神庙冒险

一款基于 Canvas 的跑酷游戏，在古埃及神庙中逃离法老的追捕！

## 🎮 游戏特色

### 4种游戏模式
- 🏃 **经典模式**：标准跑酷体验
- ⚡ **极速模式**：1.5倍初始速度，挑战反应极限
- 💗 **生存模式**：3条命，容错率更高
- 👑 **BOSS模式**：每500m触发BOSS战

### 核心玩法
- **二段跳**：空中再按跳跃键触发
- **翻滚无敌**：下滑18帧无敌时间
- **黄金狂暴**：连捡10枚金币触发x3得分
- **法老追捕**：距离过近会被抓住

### 障碍系统
- 🪨 石块、木箱、石柱
- 🔥 火焰陷阱
- ⚙️ 旋转刀盘
- 💥 激光墙（跳/滑/蹲）
- 🧱 左右墙壁突刺

### 道具系统
- 🧲 磁铁：自动吸金币
- 🛡️ 护盾：抵挡一次伤害
- ⚡ 加速：临时提速

### 排行榜
- 每个模式独立排行榜
- 本地存储最多10条记录
- 实时显示排名

## 🎨 技术亮点

- **高清渲染**：支持3倍DPR，视网膜屏完美显示
- **程序生成音乐**：Web Audio API 实时合成埃及风格BGM
- **粒子系统**：对象池优化，流畅特效
- **伪3D透视**：Canvas 2D模拟3D跑道
- **精细建模**：法老角色430行代码，骨骼/金属/布料材质分层渲染

## 🕹️ 操作说明

- **空格/↑**：跳跃（空中可二段跳）
- **↓**：下滑/翻滚
- **←/→**：切换车道
- **ESC**：暂停

## 📊 计分规则

- 距离 × 10
- 金币 × 50
- 连击（最多20）× 30
- 黄金狂暴期间 × 3

## 🚀 部署

```bash

git init
git add index.html
git commit -m "Initial commit"
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
```

然后在仓库设置中开启 GitHub Pages。
